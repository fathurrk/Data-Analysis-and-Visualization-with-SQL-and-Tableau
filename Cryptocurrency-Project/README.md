# Cryptocurrency Project
Helping the investor to know about trends of Doge Coin, so we will determine the variable of :
+ Market Capitalization
+ Volume
+ Open Price 
+ Close Price
+ High Price
+ Low Price 

### Built with :
+ PostgreSQL
+ Tableau Public
